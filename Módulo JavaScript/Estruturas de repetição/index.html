Utilizando o While

Simulando o sistema de navegação de uma nave interagindo com o usuário

  1. Pedir o nome da nave
  2. Perguntar se deseja entrar na dobra com a seguinte mensagem:

    3. "Deseja entrar na dobra espacial?"

    1- Sim

    2- Não

    se sim contabilizaremos uma dobra

  3. O usuário deve ser perguntado se deseja realizar a próxima dobra

    4. "Deseja realizar proxima dobra espacial?"

    1- Sim

    2- Não

    para cada sim contabilizaremos uma dobra

  4. A pergunta deve ser feita novamente até que a escolha seja não
  5. Ao escolher "não" em qualquer pergunta a contagem deve ser encerrada
  6. O programa deve informar o nome da nave e o Número de seguidos que a dobra foi realizada
